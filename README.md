# Spyder Table — Прототип детекции людей за столиками

## Установка

### 1. Установить зависимости через `uv`

```bash
uv sync
```

 🛠 Установка uv (если ещё не установлен)

```bash
pip install uv
# или
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Подробнее: <https://docs.astral.sh/uv/>

## Текст

- Сделал параметр запуска --video где можно указать путь до файла
- Выбрал 2 видео и 15 столик
- использовал модель YOLOv8n
- Есть два состояния: занятый и свободный. Фиксация событий: 
cleared: переход из "занят" → "пуст" (после чего стол остаётся пустым ≥20 сек)
approached: переход из "пуст" → "занят" после периода отсутствия ≥20 сек
- так как модель на данном видео иногда мелькала что видит человека то не видит, попытался сделать некое сглаживание чтобы состояние не менялось каждый кадр (текущие настройки это 60% кадров за 10 секунд)
- среднее время вышло 160 сек
- нет проблемного скриншота, но когда человек на видео начинает собираться модель его не видит (возможно потому что лицо/голова прячется за плафон)
